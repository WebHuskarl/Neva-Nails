# Neva Nails — Премиум студия маникюра

Это репозиторий лендинга премиум-студии маникюра "Neva Nails" в Санкт-Петербурге. Проект разработан с акцентом на современный, чистый и конверсионный дизайн (Awwwards-уровень) без использования тяжелых JS-фреймворков.

## Структура проекта (БЭМ Архитектура)

Проект построен на чистом HTML, CSS (по методологии БЭМ) и Vanilla JS (ES6 модули). 

```text
project/
├── index.html           # Главный и единственный HTML-файл (SPA-like landing)
├── css/
│   ├── variables.css    # Глобальные токены (шрифты, цвета, отступы, анимации)
│   ├── normalize.css    # Сброс базовых стилей
│   ├── global.css       # Общие стили для типографики, кнопок и базовых layout'ов
│   ├── media.css        # Точка сборки для медиа-запросов (адаптив)
│   └── components/      # Изолированные БЭМ-компоненты для каждой секции
│       ├── header.css, hero.css, trust.css, services.css, etc.
├── js/
│   ├── main.js          # Точка входа JS (настройка наблюдателей, скролла)
│   └── (modules)        # Модули логики: gallery, configurator, etc.
├── images/              # Графические ресурсы, оптимизированные для web (webp/png/jpg/mp4)
│   ├── hero/            # Видео и слайдер для первого экрана
│   ├── logo/            # Логотипы и favicon
│   ├── services/        # Изображения для услуг (low-poly эффекты и hover'ы)
│   └── (brands)         # Логотипы брендов для бегущей строки (OPI, CND, etc.)
└── fonts/               # Локальные файлы шрифтов (если используются)
```

## Функционал секций (index.html)

1. **Header (`<header class="header">`)**
   Закрепленное меню навигации, логотип и кнопка записи. Поддерживает бургер-меню для мобильных устройств.

2. **Hero (`<section class="hero">`)**
   Главный экран с видео-подложкой (`hero.mp4`) и динамическим текстом. Содержит плавающий интерактивный бейдж (вращающийся текст). Видео бесшовно зациклено.

3. **Trust & Stats (`<section class="trust">`)**
   Секция социального доказательства: анимированные счетчики ("лет на рынке", "мастеров", "клиентов") и бесшовная, идеально зацикленная `CSS-marquee` (бегущая строка) с логотипами используемых премиум-брендов.

4. **Portfolio / Gallery (`<section class="gallery">`)**
   Портфолио работ с возможностью фильтрации по категориям (Маникюр, Педикюр, Дизайн и т.д.).

5. **Masters (`<section class="masters">`)**
   Карточки мастеров с информацией о грейде (Топ, Senior), стаже и ссылками на отзывы/портфолио.

6. **Configurator / Booking (`<section class="configurator">`)**
   Калькулятор стоимости и форма быстрой записи. При выборе услуг динамически подсчитывается итоговая цена и время процедуры.

7. **Services & Offers (`<section class="services">`)**
   Визуализированный прайс-лист. Плашки услуг имеют hover-эффекты (переход от low-poly текстур к реальному фото) с динамическим "бликом", реагирующим на движение курсора мыши.

8. **Loyalty & Reviews (`<section class="loyalty">` / `<section class="reviews">`)**
   Информация о системе лояльности (кэшбек, скидки постоянным клиентам) и реальные отзывы.

9. **Footer (`<footer class="footer">`)**
   Подвал сайта с контактной информацией, ссылками на социальные сети, расписанием работы и юридической информацией.

## Технические правила и деплой
- **GitHub Pages Ready**: Проект полностью совместим с развертыванием на GitHub Pages. Все пути к изображениям и файлам относительные, а имена файлов приведены в нижний регистр и используют дефисы (например, `bio-sculpture.png`) для предотвращения ошибок 404 на Linux-серверах (case-sensitive).
- Зависимости отсутствуют. Для запуска достаточно открыть `index.html` в любом современном браузере или использовать Live Server.

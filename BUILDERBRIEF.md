<design_direction>
Направление D — «Минерал / Перламутр». Тёмная каменная база, перламутровый текст, лиловый неон-акцент, минеральный тиловый highlight.

Шрифты (self-host, кириллица обязательна, font-display: swap):
- Display: Geologica (variable) — заголовки H1-H3, крупная типографика.
- Body: Golos Text — параграфы, UI, кнопки.
- Mono-метки: JetBrains Mono — индексы секций [001], CASE/02, цифры, теги.
НЕ использовать Inter/Arial/Roboto/Oswald.

Токены цвета (semantic, для variables.css):
  --color-onyx:        #15161A;  /* фон-база */
  --color-onyx-raised: #1D1E24;  /* приподнятые поверхности/карточки */
  --color-graphite:    #2A2C34;  /* hairline-линии, бордеры */
  --color-pearl:       #E8E4DE;  /* основной текст / перламутр */
  --color-pearl-dim:   #A9A6A0;  /* приглушённый текст */
  --color-lilac:       #B9A0FF;  /* основной неон-акцент */
  --color-lilac-deep:  #7C5CFF;  /* пара для градиента */
  --color-mineral:     #58E0C4;  /* минеральный тиловый highlight (дозированно) */

СИГНАТУРНЫЙ ЭЛЕМЕНТ (одна запоминаемая деталь):
«Минеральный срез + перламутровый блик». Секции разделяются тончайшими перламутровыми «прожилками» (veined hairline-линии, анимированный градиент по hairline). Ключевой элемент — перламутровая капля-блик: переливающийся (CSS conic/animated gradient pearl) акцент на CTA, на активной карточке и за курсором в hero (spotlight). Bento-сетки выглядят как срез минерала.

Обязательные анти-слоп детали:
- Линейные иконки Lucide (НЕ эмодзи).
- БЕЗ тире (—, –) — только дефис (-).
- Mono-метки и индексы секций: [001], CASE/02, FIG.03, (C) 2026.
- Hairline-линии (1px перламутр/графит), угловые маркеры на карточках.
- Гигантские outline/ghost-числа на фоне секций.
- Лёгкая grain/noise-текстура поверх фона.

Анимации (со вкусом, уважают prefers-reduced-motion):
- counter-up на цифрах (Trust-бар, лояльность);
- reveal-stagger при скролле (CSS scroll-driven: animation-timeline: view());
- hover-lift на карточках, перламутровый glow на hover;
- spotlight за курсором в hero;
- marquee в отзывах/trust-баре;
- вращающийся pearl-badge в hero.
</design_direction>

<tech_stack>
- HTML5 семантический (header, main, section, article, footer, nav, form).
- CSS3 + BEM. Значения в rem (clamp() для типографики/отступов). CSS-переменные/токены.
- Vanilla JavaScript ES6 modules (import/export). JS цепляется через data-атрибуты (data-*), не через классы/id ради логики.
- ЗАПРЕЩЕНО: React, Vue, Bootstrap, Tailwind, jQuery, любые CSS-фреймворки.

Лучшие практики (июнь 2026), применить:
- CSS scroll-driven animations (animation-timeline: view()/scroll()) для reveal.
- View Transitions API для модалки портфолио и плавных переходов (с фолбэком).
- :has(), container queries (@container), color-mix(), text-wrap: balance, clamp().
- Liquid Glass через backdrop-filter (дозированно: хедер, активные карточки) с фолбэком.
- Доступность: ARIA-роли, видимый focus-visible, контраст AA, клавиатурная навигация, landmarks.
- Производительность: self-host шрифтов, lazy-load изображений (loading="lazy", decoding="async"), srcset/sizes для адаптивных фото, минимизация CSS, prefers-reduced-motion.
</tech_stack>
